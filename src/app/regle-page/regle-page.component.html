Instructions du jeu

Le but du jeu est de deviner un Pokémon, un objet ou une capacité compris entre la première génération et la neuvième génération.

À chaque essai, il vous est indiqué des informations sur ce que vous cherchez par rapport à votre entrée. Par exemple, si le Pokémon que vous chercher est plus lourd ou pas que celui que devez deviner. Le nombre d'essais varie pour chaque catégorie.

Les couleurs des cases vous donnent des indices également :

  • Vert : l'information est juste. Ce que vous avez entré et ce que vous cherchez partagent cette information.
  • Orange : l'information est partiellement juste.
  • Rouge : l'information est incorrecte.

Supposons que le Pokémon que vous cherchez est Reptincel.

Si vous entrez Salamèche, la colonne Types s'affichera en vert : ils ont le même type.

 Si vous entrez Dracaufeu, la colonne Types s'affichera en orange : ils ont un type en commun, mais l'autre est faux.

 Si vous entrez Bulbizarre, la colonne Types s'affichera en rouge : aucun type de Bulbizarre et en commun avec ceux de Reptincel.

Devinez un Pokémon

Le but du jeu est de deviner un Pokémon, un objet ou une capacité compris entre la première génération et la neuvième génération. Vous pouvez entrer le nom du Pokémon en français (sans oublier les accents) ou son numéro dans le Pokédex.

  • Les Méga-Évolutions ne sont pas incluses, ainsi que les formes régionales.
  • Deux Pokémons simple type ont un type en commun : le type "inexistant". Par exemple, si vous devez trouver Salamèche et que vous rentrez Carapuce, on vous indiquera un type en commun.

Devinez un objet

Les matériaux en 9G, les CT/CS, les ingrédients, les décorations de pique-nique, les cristaux Dynamax, les objets inutilisables et objets qui font avancer les histoires des jeux ont été retirés. Seuls les noms en anglais sont disponibles.


Devinez une capacité

Toutes les capacités sont présentes. Seuls les noms anglais fonctionnent.
